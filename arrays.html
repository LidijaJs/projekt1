Polja


Polja JavaScript se uporabljajo za shranjevanje več vrednosti v eni spremenljivki.
Primer:
var avtomobili = ["Saab", "Volvo", "BMW"];

Kaj je polje?
Polje je posebna spremenljivka, ki lahko vsebuje več kot eno vrednost hkrati.
Če imate seznam elementov (na primer seznam imen avtomobilov), bi lahko shranjevanje avtomobilov v posamezne spremenljivke izgledalo takole:
var avto1 = "Saab";
var avto2 = "Volvo";
var avto3 = "BMW";

Kaj pa, če želite zapeljati skozi avtomobile in najti točno določenega? In kaj, če ne bi imeli 3 avtomobilov, ampak 300?
Rešitev je polje! Polje lahko vsebuje več vrednosti pod enim imenom, do vrednosti pa lahko dostopate s sklicevanjem na indeksno številko.

Ustvarjanje polje
Uporaba dobesednega polja je najlažji način za ustvarjanje JavaScriptnega polja.
Sintaksa:
var polje_ime = [predmet1, predmet2, ...];

Primer:
var avtomobili = ["Saab", "Volvo", "BMW"];
Razmiki in prelomi vrstic niso pomembni. Izjava lahko obsega več vrstic:

Primer:
var avtomobili = [
"Saab",
"Volvo",
"BMW"
];

Uporaba ključne besede JavaScript "novo"
Naslednji primer prav tako ustvari polje in ji dodeli vrednosti:
Primer:
var avtomobili = novo polje ("Saab", "Volvo", "BMW");

Zgornja primera počneta popolnoma enako. Ni potrebe po uporabi "novo Polje ()". Za poenostavitev, berljivost in hitrost izvajanja uporabite prvo (metoda dobesednega polja).

Dostop do elementov polja
Do elementa polja dostopate tako, da se sklicujete na indeksno številko. Ta izjava dostopa do vrednosti prvega elementa v avtomobilih:
var ime = avtomobili [0];

Primer:
var avtomobili = ["Saab", "Volvo", "BMW"];
document.getElementById ("demo"). innerHTML = avtomobili [0];

Opomba: Indeksi nizov se začnejo z 0. [0] je prvi element. [1] je drugi element.

Spreminjanje elementa polja
Ta izjava spremeni vrednost prvega elementa v avtomobilih:
avtomobili [0] = "Opel";

Primer:
var avtomobili = ["Saab", "Volvo", "BMW"];
avtomobili [0] = "Opel";
document.getElementById ("demo"). innerHTML = avtomobili [0];

Vaja 1/3.
Ustvari polje in mu dodaj dve vrednosti. Polje nato izpiši.

Vaja 2/3.
Ustvari polje z dvema elementoma in izpiši njegovo dolžino.

Vaja 3/3.
Ustvari polje z dvema elementoma in nato izbriši zadnji element.





Dostopite do celotnega polja
Z JavaScriptom lahko do celotnega polja pridete tako, da se sklicujete na ime polja:
Primer:
var avtomobili = ["Saab", "Volvo", "BMW"];
document.getElementById ("demo"). innerHTML = avtomobili;

Lastnosti in metode polj
Prava moč nizov JavaScript so vgrajene lastnosti in metode polj:
Primeri:
var x = avtomobili.length; // Lastnost length vrne število elementov
var y = avtomobili.sort (); // Metoda sort () razvršča polja
Metode nizov so zajete v naslednjih poglavjih.

Lastnost dolžina
Lastnost dolžine polja vrne dolžino polja (število elementov polja).
Primer:
var sadje = ["Banana", "Orange", "Apple", "Mango"];
sadje.dolžina; // dolžina plodov je 4

Dostop do prvega elementa polja
Primer:
sadje = ["Banana", "Orange", "Apple", "Mango"];
var first = sadje [0];

Dostop do zadnjega elementa polja
Primer:
sadje = ["Banana", "Orange", "Apple", "Mango"];
var last = sadje [sadje.dolžina - 1];

Looping elementov polja
Najvarnejši način zanke skozi polje je uporaba zanke za (for):
Primer:
var sadje, besedilo, fLen, i;
sadje = ["Banana", "Orange", "Apple", "Mango"];
fLen = sadje.dolžina;

text = "<ul>";
za (i = 0; i < fLen; i ++) {
text + = "<li>" + sadje [i] + "</li>";
}
text + = "<ul>";

Uporabite lahko tudi funkcijo Array.forEach ():

Primer:
var sadje, besedilo;
sadje = ["Banana", "Orange", "Apple", "Mango"];
text = "<ul>";
sadje.forEach (myFunction);
besedilo + = "</ul>";

funkcija myFunction (vrednost) {
besedilo + = "</li>" + vrednost + "<//li>";
}

Dodajanje elementov
Najlažji način za dodajanje novega elementa v polje je uporaba metode push ():
Primer:
var sadje = ["Banana", "Orange", "Apple", "Mango"];
sadje.push ("Limona"); // sadju doda nov element (Lemon)
Nov element lahko v polje dodate tudi z uporabo lastnosti length:

Primer:
var sadje = ["Banana", "Orange", "Apple", "Mango"];
sadje [fruit.length] = "limona"; // sadju doda nov element (Lemon)

OPOZORILO ! Dodajanje elementov z visokimi indeksi lahko ustvari nedefinirane "luknje" v polju:
Primer:
var sadje = ["Banana", "Orange", "Apple", "Mango"];
sadje [6] = "Limona"; // sadju doda nov element (Lemon)

Funkcije


Funkcija JavaScript je blok kode, zasnovan za izvajanje določene naloge.
Funkcija JavaScript se izvede, ko jo "nekaj" prikliče (pokliče).

Primer:
funkcija myFunction (p1, p2) {
vrnitev p1 * p2; // Funkcija vrne zmnožek p1 in p2
}

Sintaksa funkcije JavaScript
Funkcija JavaScript je definirana s ključno besedo funkcije, ki ji sledi ime, ki mu sledijo oklepaji ().
Imena funkcij lahko vsebujejo črke, števke, podčrtaje in znake za dolar (enaka pravila kot spremenljivke).
V oklepajih so lahko imena parametrov, ločena z vejicami:
(parameter1, parameter2, ...)

Koda, ki jo mora funkcija izvesti, je umeščena v zavite oklepaje: {}
ime funkcije (parameter1, parameter2, parameter3) {
// koda za izvedbo
}

Parametri funkcije so navedeni v oklepajih () v definiciji funkcije.
Argumenti funkcije so vrednosti, ki jih funkcija prejme ob njenem klicu.
Znotraj funkcije se argumenti (parametri) obnašajo kot lokalne spremenljivke.
Funkcija je v drugih programskih jezikih podobna postopku ali podprogramu.

Poklic funkcije
Koda znotraj funkcije se bo izvedla, ko "nekaj" prikliče (pokliče) funkcijo:
Ko se zgodi dogodek (ko uporabnik klikne gumb)
Ko je poklican (poklican) iz kode JavaScript
Samodejno (samodejno poklicano)

Vrnitev funkcije
Ko JavaScript doseže povratni stavek, se funkcija ustavi. Če je bila funkcija priklicana iz stavka, se bo JavaScript vrnil za izvajanje kode po stavku za klicanje. Funkcije pogosto izračunajo vrnjeno vrednost. Vrnjena vrednost se "vrne" nazaj klicatelju:

Primer:
Izračunajte zmnožek dveh števil in vrnite rezultat:
var x = myFunction (4, 3); // Pokliče se funkcija, vrnjena vrednost se bo končala v x

funkcija myFunction (a, b) {
vrni a * b; // Funkcija vrne zmnožek a in b
}

Rezultat v x bo: 12.

Zakaj funkcije?
Kodo lahko znova uporabite: Kodo določite enkrat in jo uporabite večkrat. Isto kodo lahko večkrat uporabite z različnimi argumenti, da dobite različne rezultate.

Primer:
Pretvori Fahrenheita v Celzijeve:
funkcija do Celzija (Fahrenheit) {
povratek (5/9) * (fahrenheit-32);
}
document.getElementById ("demo"). innerHTML = toCelsius (77);

() Operator prikliče funkcijo
V zgornjem primeru se toCelsius sklicuje na objekt funkcije, toCelsius () pa na rezultat funkcije. Dostop do funkcije brez () bo vrnil funkcijski objekt namesto rezultata funkcije.

Primer:
funkcija do Celzija (Fahrenheit) {
povratek (5/9) * (fahrenheit-32);
}
document.getElementById ("demo"). innerHTML = toCelsius;

Funkcije, ki se uporabljajo kot spremenljive vrednosti
Funkcije lahko uporabljate na enak način kot spremenljivke pri vseh vrstah formul, dodeljevanj in izračunov.

Primer
Namesto da bi uporabili spremenljivko za shranjevanje vrnjene vrednosti funkcije:
var x = do Celzija (77);
var text = "Temperatura je" + x + "Celzija";
Funkcijo lahko uporabite neposredno kot spremenljivko:

var text = "Temperatura je" + do Celzija (77) + "Celzija";

Lokalne spremenljivke
Spremenljivke, prijavljene v funkciji JavaScript, postanejo LOKALNE za funkcijo. Do lokalnih spremenljivk je mogoče dostopati samo znotraj funkcije.

Primer:
// koda tukaj NE more uporabljati avtoIme
funkcija myFunction () {
var avtoIme = "Volvo";
// koda tukaj LAHKO uporabite carName
}
// koda tukaj NE more uporabljati carName

Ker so lokalne spremenljivke prepoznane samo znotraj njihovih funkcij, lahko spremenljivke z istim imenom uporabljamo v različnih funkcijah.
Lokalne spremenljivke se ustvarijo ob zagonu funkcije in izbrišejo, ko se funkcija zaključi.

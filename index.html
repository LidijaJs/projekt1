Spremenljivke


JavaScript spremenljivke so vsebniki za shranjevanje podatkovnih vrednosti, kot na primer številka, ki jo lahko uporabimo v vsoti, ali niz, ki ga lahko uporabimo kot del stavka.

Primer:
var x = 5;
var y = 6;
var z = x + y;

Z zgornjega primera lahko pričakujemo, da:
- spremenljivka x bo shranila vrednost 5,
- y bo shranil vrednost 6.
- z bo shranil vrednost 11.

Imena spremenljivk
Vse spremenljivke JavaScript je treba identificirati z enoličnimi imeni. Ta enolična imena se imenujejo identifikatorji. Identifikatorji so lahko kratka imena (na primer x in y) ali več opisnih imen (starost, vsota, skupni volumen). Splošna pravila za oblikovanje imen za spremenljivke (enolični identifikatorji) so:

Imena lahko vsebujejo črke, števke, podčrtaje in dolarske znake.
Imena se morajo začeti s črko
Imena se lahko začnejo tudi z $ in _ (vendar ga v tej vadnici ne bomo uporabljali)
Imena razlikujejo med velikimi in malimi črkami (y in Y sta različni spremenljivki)
Rezerviranih besed (kot so ključne besede JavaScript) ni mogoče uporabiti kot imena

Identifikatorji JavaScript razlikujejo med velikimi in malimi črkami!

Operater dodelitve
V JavaScript je enačba (=) operator "dodelitve" in ne operator "je enako". To se razlikuje od algebre. Naslednje v algebri ni smiselno:
x = x + 5
V JavaScript pa je povsem smiselno: x dodeli vrednost x + 5. (Izračuna vrednost x + 5 in rezultat postavi v x. Vrednost x se poveča za 5.)

Operator " je enako" je v JavaScript zapisan kot ==.

JavaScript podatkovni tipi
Spremenljivke JavaScript lahko vsebujejo številke, kot je 100, in besedilne vrednosti, kot je "John Doe". Pri programiranju se besedilne vrednosti imenujejo besedilni nizi.
JavaScript lahko obdeluje številne vrste podatkov, ampak za zdaj si zapomnite le številke in nize. Nizi so zapisani znotraj dvojnih ali enojnih narekovajev. Številke so zapisane brez narekovajev. Če številko postavite v narekovaje, bo obravnavana kot besedilni niz.

Primer
var pi = 3,14;
var oseba = "Janez Novak";
var odgovor = 'Da, sem!';


Deklariranje (ustvarjanje) spremenljivk JavaScript
Ustvarjanje spremenljivke v JavaScript se imenuje "izjava" (ang. declaring) spremenljivke. Spremenljivko JavaScript deklarirate s ključno besedo var:
var avtoIme;
Po deklaraciji spremenljivka nima vrednosti (tehnično ima vrednost undefined).

Če želite spremenljivki dodeliti vrednost, uporabite znak enačbe:
avtoIme = "Volvo";
Spremenljivki lahko dodelite tudi vrednost, ko jo izjavite:
var carName = "Volvo";

Ponovno prijavljanje spremenljivk JavaScript
Če ponovno deklarirate spremenljivko JavaScript, ta ne bo izgubila vrednosti. Spremenljivka carName bo po izvedbi teh stavkov še vedno imela vrednost "Volvo":
var carName = "Volvo";
var carName;

JavaScript aritmetika
Kot pri algebri lahko tudi s spremenljivkami JavaScript naredite aritmetiko z uporabo operatorjev, kot sta = in +:
Primer
var x = 5 + 2 + 3;

Posebnost pri spremenljivkah je, da se njihove vsebovane vrednosti lahko spreminjajo.
Druga posebnost spremenljivk je ta, da lahko vsebujejo skoraj vse - ne le nizov in števil.
Spremenljivke lahko vsebujejo tudi zapletene podatke in celo celotne funkcije.
